# Hindi Fluency Analysis (VFT + SpAM)

This project runs a full Hindi fluency analysis pipeline using:
- VFT (Verbal Fluency Task)
- SpAM (Spatial Arrangement Method)
- Transformer-based semantic and phonetic clustering
- BRSM-aligned statistical tests

## Files
- `analysis_pipeline.py`: main analysis script
- `merged_vft_spam_responses_enriched.csv`: processed dataset
- `responses.json`: raw responses
- `Report_Final.tex`: LaTeX paper
- `Report_Final.md`: markdown report generated by pipeline
- `images/`: generated figures

## 1. Setup

### Prerequisites
- Python 3.10+
- `pip`
- (Optional, for LaTeX PDF) MiKTeX / TeX Live with `pdflatex`

### Create and activate a virtual environment (PowerShell)
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### Install dependencies
```powershell
pip install -r requirements.txt
```

## 2. Run the analysis pipeline

```powershell
python analysis_pipeline.py
```

This generates/updates:
- `Report_Final.md`
- `Report_Final_md.pdf` (markdown-export PDF)
- figures under `images/`

## 3. Generate the final LaTeX paper PDF

`Report_Final.pdf` is the LaTeX-based final paper.

Run:
```powershell
pdflatex -interaction=nonstopmode -halt-on-error Report_Final.tex
pdflatex -interaction=nonstopmode -halt-on-error Report_Final.tex
```

This creates/updates:
- `Report_Final.pdf` (authoritative LaTeX output)

## 4. Typical workflow
1. Run `python analysis_pipeline.py` to refresh data-driven tables/figures.
2. Update `Report_Final.tex` text if needed.
3. Compile LaTeX twice to refresh `Report_Final.pdf`.

## Notes
- First run may download Hugging Face model weights for `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2`.
- If you see rate-limit warnings from Hugging Face, set `HF_TOKEN` in your environment.
